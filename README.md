# Шахматы на Python с графическим интерфейсом

Курсовая работа по разработке простой шахматной игры с использованием Python и библиотеки Tkinter.

## Особенности

- Полноценная шахматная доска 8×8
- Поддержка всех фигур: пешка, ладья, конь, слон, ферзь, король
- Валидация ходов (включая защиту от шаха)
- Превращение пешки в ферзя при достижении последней горизонтали
- История ходов с отображением в интерфейсе
- Возможность отменить последний ход и начать новую игру
- Юникод-символы для отображения фигур
