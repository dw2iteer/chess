# Шахматы (Chess)
Это десктопная игра, написанная на Python с использованием библиотеки tkinter.
Цель игры — сыграть партию в шахматы по классическим правилам на локальной машине.

# Возможности
Полноценная графическая доска 8×8 с интерактивными фигурами
Поддержка ходов всех фигур (пешки, ладьи, кони, слоны, ферзь, король)
Проверка правильности ходов согласно классическим шахматным правилам
Индикация шаха и мата
Возможность отмены хода (если реализовано)
Простой и понятный интерфейс на русском языке
# Требования
Python 3.8+
Библиотека tkinter (входит в стандартную поставку Python)
